8.2 WARNING MESSAGES

no mas.ini - defaulting include/ and device/
mas.ini file have two lines. First line is include file path, and second is path to device directory. By default it is in include/ and in device/ directory. If you see this message, just write new ini file with those two lines.

can not remove temporaly file
temporaly file is created for communication beetwen two passes. This file can not be deleted (possible BUG)

identifer too long - truncating
name is too long, it will be truncated to 30 chars (or something like this)

redeclaration of macro ???
for now it is warning, but it will be error in future releases. It means that macro with same name already exist

to few parameters in macro ???
macro was called with more params than it had

expected ??? bits num, truncating from ??? to ???
number(operand) is truncated so it will be in size that can fit into opcode. This message is shadowd by device file, becuase every instruction can define it's message for numbers that is greater than opcode fild
