7.3 eeprom.inc

This file has only one macro "Declare". This macro declare 'n' numbers in eeprom. Before using this file it must be specified device 'eeprom8' or 'eeprom16'. 
  .device eeprom8

  .include <eeprom.inc>
  start:
   Declare(0,1,2,3,4,5,6,7)
   Declare(0,2,4,6,8,10,12,14)
   Declare(1,3,5,7,9,11,13,15)

Device eeprom8 and eeprom16 implement only one instruction: 'decl'. It has one parametar that is 8/16bit data: 

  .device eeprom8

  start:

   decl 0

   decl 1

   decl 2

   decl 3

   decl 4
This example declares 5 numbers (0,1,2,3 and 4) in eeprom.
