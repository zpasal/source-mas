5.7 .rep .endrep

Directives used for repetition: 
   .rep expression

      ; code

   .endrep
.rep is not like 'while' in high-languages. .rep calculate expression, after this, repetitions goes "expression" times, so this is not what programmer means: 
  b = 20

  .rep b

     movlw b

     b = b - 2   ; no effect for
.rep

  .endrep
This loop will go 20 times, not 10 times!

DO NOT WRITE CODE LIKE UPPER EXAMPLE. ON NEXT RELEASE OF MAS, UPPER VARIABLE 'b' WILL CHANGE REPETITION COUNT!!!!
