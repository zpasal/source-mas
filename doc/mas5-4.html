5.4 .include

.include does what it says - include other file into source code. Syntax for .include is like C/C++ #include: 
  .include <something> ; include file from standard
inc. directiry

  .include "something"    ; include file
from current directory

   

  .include <p16c84.inc>

   start: 

        movf 0x10, W  
; W from p16c84.inc
