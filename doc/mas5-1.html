5.1 .org

.org evaluate current offset to new value. Default value is 0: 
  start: goto dalje

         .org 0x100

  dalje:
In this case, current-offset at start is 0, so label 'start' is at address 0x0000 and label 'dalje' is at adress 0x0100.

In some output formats (see '6. Output file formats') all memory that is skipped by .org is filled with nops. Some other formats do not require this. .org can not have operand that is expression, only const number can be used.
