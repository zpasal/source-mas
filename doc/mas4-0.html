4. ASSIGN EXPRESSION

With declaration of label we introduce new name into compilation. Labels have type L_LABEL. This is just intern type, true type is 32bit signed integer, or correctly:

   sizeof(signed int)

Types are:

  • L_LABEL - label
  • L_VAR - variable
  • L_CONST - constant
  • L_MACRO - macro definition


Some types have restrictions (we will see this later).
