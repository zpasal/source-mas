1. WHAT IS MAS-PIC
Mas is generic assembler for PICxxx family of microcontrollers that can generate code for more than 90 different icrocontrollers.

Mas is 2 pass assembler for Microship's (*) PICxxx family of microcontrollers. Futures:
 

  • about 90 microcont. and eeprom8/16 supported
  • 2 pass assembly
  • conditional assembly
  • simple, but powerful macros
  • different file formats for output
  • easy configuration for another PICxxx families (generic assembler devices)
  • compilation speed
  • standard include files that can save some time while programming
  • highly portable across platforms that supports C++
  • GPLed license :)
Microchip is trademark of Microchip, Inc.
