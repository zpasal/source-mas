2. COMMAND SINTAX
Usage: mas [option(s)] file

options:
-h Help screen
-v Dump version info
-f'format' Format of output file:
    'bin'
    'bt'
    'hex' (see Output file formats)
    Default is 'hex'
  -o'file'  Name of output file. If omitted, output file have same name as input but extension is .bt/.bin/.hex.
  -s'file'  Symbol file
  -d'device' Dump specification for particular device
  -I'path'  Include directory path
  -i'path'  Devices directory path

Default include and device directory path are in file mas.ini. This file is writen uppon installation. Swithches -I and -i override this settings.
